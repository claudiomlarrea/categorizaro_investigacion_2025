# Valorador de CV Docente – UCCuyo

Archivos incluidos:
- app.py — Aplicación principal en Streamlit.
- criteria.json — Configuración de secciones e ítems.
- patterns.json — Patrones de reconocimiento.
- README.txt — Guía rápida.

## Cómo usar
1. Instalar dependencias:
   pip install streamlit python-docx pandas
2. Ejecutar:
   streamlit run app.py
3. Subir un CV en formato .docx.
